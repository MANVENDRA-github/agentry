description: Produce a session handoff note for the current working session.
---

Apply the `session-handoff` skill to produce a handoff note covering the current session.

$ARGUMENTS, if provided, is guidance about what the handoff should emphasize — specific decisions to highlight, files the next session should look at first, or known gotchas worth pulling forward in the note.

If the session's work finished cleanly and the PR description carries the context, the skill's own guidance is to recommend skipping the handoff. Do that rather than producing low-value ceremony.
